DX Shop 
Projeto de desenvolvimento de uma lojinha em ReactJS.
Preview em:
https://jonathantoyoshima.github.io/dxshop/#/

Prerequisitos:
- Node
- npm
- vscode (ou outro IDE que possa entender o jsconfig.json)

Instalando:
```
 - npm install
```
Desenvolvimento:
 ```
 - npm start
```

Testando:
```
- npm run test
```

Build:
```
 - npm run build;
```

Deploy:
```
 - npm run deploy;
```

 Entrega no git.pages.
